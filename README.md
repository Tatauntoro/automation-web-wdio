## Running Tests

Clone Repository

```sh
$ git clone git@github.com:Tatauntoro/automation-web-wdio.git
$ cd automation-web-wdio
```

Install dependencies:

```sh
$ npm install
```

Setup Environment

```sh
copy value from .env.sample to .env
```

To run order creation, run:

```sh
$ npm run order
```

To run order monitoring, run:

```sh
$ npm run checkOrder
```
